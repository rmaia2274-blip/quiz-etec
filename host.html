Configure o Firebase antes de começar.
1. Acesse console.firebase.google.com → crie um projeto → adicione um app Web.
2. Em Realtime Database → crie o banco → copie a URL (https://SEU-PROJETO.firebaseio.com).
3. Em Rules, cole: { "rules": { ".read": true, ".write": true } } (para testes).
4. Preencha os campos abaixo e clique em Salvar.
